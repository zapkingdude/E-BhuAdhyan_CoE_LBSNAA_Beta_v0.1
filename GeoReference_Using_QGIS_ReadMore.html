About Module

Georeferencing Using QGIS: Simple Starter Steps to Help in Geo-Referencing

Geo-Referencing is the main and important part of the Geospatial data processing and which helps in bringing the non spatial data into a spatial data

Module Topics
Module Overview

This E-Module, “Georeferencing Using QGIS: Simple Starter Steps to Help in Geo-Referencing”, serves as a beginner-friendly guide to the essentials of georeferencing. It introduces you to the step-by-step process using the open-source GIS software, QGIS. The content has been carefully compiled and structured to ensure clarity and simplicity, making it easy for learners to follow along and perform the georeferencing process independently.

Learning Objectives

Upon completion of this module, learners will be able to:

Understand the Meaning and Purpose of Geo-Referencing in GIS

Recognize the Different types of Geo-Reference

Understand the Steps Available in Performing all the Types of Geo-Referencing

Gain proficiency in using QGIS software tools for Geo-Referencing

1. Introduction

Introduction to Geo-Referencing

2: Geo-referencing using Ground Control Points (GCPs):

Steps involved in the geo-referencing using the GCPs method.

3. Geo-referencing Using Image-to-Image:

Steps involved in the geo-referencing using the Image-to-Image method.
