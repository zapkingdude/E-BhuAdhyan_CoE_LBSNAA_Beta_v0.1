Welcome to Chapter 1 : Introduction

Understanding the concept

1. Introduction

The creation of property layers from ground truthing points is a critical step in modern geospatial analysis and land management.

This E-Module focuses on leveraging data collected from GNSS NRTK Rovers—advanced Global Navigation Satellite System devices that provide real-time kinematic positioning with high accuracy. By gathering precise ground truthing points directly from the field, these rovers enable the accurate delineation of property boundaries and features.

The module guides users through the process of transforming raw GNSS NRTK data into comprehensive digital property layers, which can be integrated into Geographic Information Systems (GIS) for enhanced decision-making, planning, and resource management.

Understanding this process is essential for professionals in surveying, urban planning, environmental monitoring, and related fields who aim to improve spatial data quality and reliability through ground-based verification.

1.2 Brief About GNSS NRTK Surveying

When surveyors go to the field to mark property boundaries, accuracy is very important. Even small errors can cause disputes in land ownership and records. This is where GNSS NRTK Rovers play a vital role.

GNSS (Global Navigation Satellite System) is the technology behind GPS. It uses signals from satellites to identify a location on the Earth’s surface. Normally, GPS on a mobile phone is accurate only up to a few meters, which is not enough for property surveys. But NRTK (Network Real-Time Kinematic) technology improves this accuracy to the centimeter level by using correction signals from a reference network.

  • Fetches precise ground points from the field in real time

  • Marks corners and edges of plots with very high accuracy.

  • Saves data digitally, which can later be processed in GIS software to create property layers.

This process of using rovers to collect exact coordinates is called ground truthing—it confirms the real position of features on the ground. Once collected, these points are joined together to form polygons, representing the true shape and size of properties on a digital map.

In short, GNSS NRTK Rovers help surveyors turn field observations into accurate digital property maps, making land administration faster, reliable, and transparent.

Welcome to Chapter 2 : Steps for Propery Layer Creation using QGIS

Step by Step process explained for the creation of property layers - Utilizing the tool available in QGIS Software

2.1. Introduction

QGIS (Quantum GIS) provides powerful tools for georeferencing raster images (e.g., scanned maps, aerial photos) to align them with real-world coordinates. The Georeferencer tool in QGIS allows for multiple techniques, each suitable for different types of spatial distortions.

Things to keep ready before starting the step-by-step process:

  • QGIS Software installed and opened on the computer system. - Software may download QGIS from the link - QGIS - Download
  • Open the installed QGIS Application.
  • CSV / Exported data from GNSS NRTK Rovers : The data collected in the field using the GNSS NRTK Rovers are to exported in either in CSV or any other supported vector formats.
  • Ortho Rectified Image (ORI) : A good quality ORI basemap which helps in better understanding on clearer property layers generation

Process Flow:

2.2. Steps for Propery Layer Creation using QGIS

Following are the steps involved in the process of creating the propery layers using the tools in the QGIS software.

1. Open QGIS and start a new project
Save a Project File in QGIS
  • Go to View    Project    Save As.
  • Enter the project name and then click "Save"" to save the project with the extension *.qgz
2. Importing CSV to QGIS and creating Property Boundary Point. (CSV)
Ensure your CSV/Excel file contains spatial data (latitude and longitude or X, Y coordinates). Open the file in Microsoft Excel and check for:
  • Proper column names (e.g., Latitude, Longitude or Easting, Northing). Consistent data format (numbers should not have text values).
  • No empty or null values in coordinate columns.
  • To Import CSV File:
    • Click on Layer    Add Layer    Add Delinited Text Layer.
  • The Data Source Manager Delimited text window will pop up
  • In the Data Source Manager window:
    • Click on Browse     Select your CSV File
    • Browse your path and select the CSV file
  • Choose Point coordinates as Geometry Definition.
  • Assign the X Field (Longitude/Easting) and Y Field (Latitude/Northing). Set the correct Coordinate System (e.g., EPSG:32644 ).
  • Click Add and then Close – the points will now appear on the map, and the CSV layer will be added to the Layers panel.
3. Exporting the Property Boundary Points (csv) to shape file
  • Right-click on the imported CSV layer in the Layers panel.
    • Navigate to Export    Save Feature As on the Menu Bar as shown in the below images
  • In the Save Vector Layer As window:
    • Set Format to ESRI Shape file
    • Click Browse and select a folder to save the shape file. Eg: Propery_Boundary_points
    • Choose an appropriate Coordinate Reference System (CRS).
    • Click OK to create the shape file.
    • The shape file will be automatically added to the Layers panel.
4. Creation of Property Parcel polygon Shape file.
To create polygons, you need to create a Shape file Polygon Layer
  • Navigate to Layer    Create Layer    New Shape file Layer
  • Enter a File Name.(Eg: Propety_Parcel) -> Choose Polygon as the Geometry Type.
  • Set the Coordinate Reference System (CRS), EPSG: 32644. -> click ok.
5. Digitization of Property Parcel Polygon
  • Now, we need to create polygons using the collected survey points, which were gathered from the field and exported as a shape file.
  • Now we need to add the fields to the Property_parcel.shp file
  • For adding field, we have Enable the Snapping Tool
  • Click Toggle Editing Mode (pencil icon) on the Digitizing Toolbar.
  • Double-click on Propery_Parcel.shp
  • Go to Symbology > Select Simple Line > Choose the Color of your choice
  • Ensure the Simple Line option is selected.
  • Click on Toggle Editing to enable the editing options.
  • Then, use the Add Polygon Feature tool to draw the polygon by manually adding vertices at the desired points
  • You have to add vertices for different points.
  • After completing the polygon, right-click to finish it.
  • Enter an ID in the attribute table and click OK.
  • Click on Save Layer Edits, then click on Toggle Editing again to stop editing.
  • In this way, you can add all the polygons.
6. Adding and editing the field and its attribute.
  • Enable Editing Mode : Right-click on the Propety_Parcel.shp in the Layers Panel.
  • Select Toggle Editing to enable editing mode.
  • Open Attribute Table : Right-click on the layer again and select Open Attribute Table and click on the newfield
  • In the Add Field window:
    • Enter Name: ULPIN
    • Type: Integer (Short 32)
    • Click OK.
    • Now, the new field ULPIN will appear in the attribute table.
  • Edit Attribute Values:
    • Click on the Pencil icon at the top of the attribute table to enable editing.
    • Click on any cell to edit its value directly.
    • After editing the attribute, the following window will appear.
  • Click on Save Edits to save the changes made to the attribute table.
7. Conclusion
The property parcel layer created from the imported from property boundary points(CSV) files will be integrated with the Land Parcel data. This integration enables linking property tax and RoR (Record of Rights) details with the corresponding land parcels. This approach enhances data accuracy and supports efficient land administration.

Welcome to Chapter 3 : Geo-referencing Using Image-to-Image

3.1. Introduction

Image-to-Image geo-referencing is the process of aligning one image (such as a scanned map or satellite image) with another image that already has real-world coordinates. Instead of using ground control points from the field, you pick matching points on both images (like road intersections, rivers, or landmarks) and link them together.

This way, the “unknown” image inherits the geographic reference from the “known” image, making it location-ready and usable for measurement, analysis, or overlay with other GIS datasets.

Things to keep ready before starting the step-by-step process:

  • QGIS Software installed and opened on the computer system.
  • Open the installed QGIS Application.
  • Target Image or Raster data: On which the Geo-Referencing has to be done.
  • Reference image using which the georeferencing is to be done.

3.2. Steps for Georeferencing with Image-to-image in QGIS

1. Open QGIS and Load Reference Data
  • Open QGIS and add the georeferenced raster image (reference image) by navigating to QGIS    Layer    Add Layer    Add Raster Layer... on the Menu Bar as shown in the below images

  • Set Projection Co-ordinate Reference System (CRS) as per the requirement

2. Open the Georeferencer Tool
  • Navigate to QGIS    Layer    Georeferencer...  on the Menu Bar as shown in the below image.
  • Click on the Georeferencer... link to open the tool to be used for georeferencing of the image
3. Load Source Raster (Unreferenced Image)
  • On the opened Georeferencer tool, click Open Raster... and select the scanned map or aerial image, etc., which is to be georeferenced, as shown in the below image
4. Add Control Points
  • Click Add Point and select recognizable locations on the unreferenced image
  • In the Enter Map Coordinates window, click "From Map Canvas".
  • Select the same location in the reference image.
  • Repeat this for multiple control points (at least 4 for better accuracy).
5. Choose the Transformation Method
  • Click Transformation Settings... by navigating to Settings    Transformation Settings...  on the Menu Bar of the Georeferencer window as shown in the below image to open the "Transformation Settings" window
  • In the opened Transformation Settings window, select a transformation type:
    • Affine Transformation (Minimum 3 GCPs; for basic scaling, rotation, translation).
    • Polynomial (1st or 2nd Order) (For slight distortions; requires 6+ GCPs).
    • Thin Plate Spline (Rubber Sheeting) (For flexible warping; good for historical maps).
    • Projective Transformation (For perspective distortions; requires 4+ GCPs).
6. Set Output Parameters
  • Choose:
    • Output format: GeoTIFF (recommended).
    • Coordinate Reference System (CRS): Choose a CRS based on the project requirements
    • Enable Load in QGIS when done.
7. Start Georeferencing
  • Click "Start Georeferencing" on the Tool Bar Menu of the "Georeferencer" window to start the georeferencing process.
  • QGIS will generate a georeferenced raster.
8. Verify Alignment
  • Overlay the newly georeferenced raster with the reference layer.
  • If misaligned, adjust control points and re-run.

Welcome to Chapter 4 : Cool Things You Can Do With GIS

4.1. Introduction

Once you understand how GIS works, you’ll realize that its possibilities are endless. GIS is not just for geographers — it’s for everyone who wants to make smarter decisions using location information.

Some cool applications:

  • Shortest Route Calculation: Delivery apps like Zomato, Uber, and Amazon use GIS to find the best routes. They calculate distances, traffic conditions, and even weather to optimize deliveries.
  • Urban Planning: Cities use GIS to decide where to build new schools, parks, and hospitals. They analyze population densities, land prices, road networks, and existing facilities to make smarter plans.
  • Wildlife Protection: Conservationists map forest cover, track animal movements, and identify areas under threat. GIS helps in designing wildlife corridors and protecting endangered species.
  • Disaster Management: During floods, earthquakes, or fires, GIS provides real-time maps showing affected areas, shelters, and safe evacuation routes. It helps save lives by giving quick, accurate information.
  • Climate Change Studies: Scientists use GIS to study how glaciers are melting, how rainfall patterns are changing, and how sea levels are rising. GIS helps visualize slow changes over decades.

4.2. Create Smart Maps & Analyze Spatial Data

4.2.1. Create Smart Maps:

GIS enables the layering of various kinds of geographic information into a single, unified visual display. These smart maps are dynamic, they don't just show roads or rivers but can also include population density, crime rates, or soil types depending on the purpose. This layered approach makes it easier to see relationships, patterns, and trends that might otherwise go unnoticed.

For example, during urban development, a smart map can show where population is growing, how close areas are to public utilities, or which regions have environmental restrictions. These insights help decision-makers take smarter, more sustainable actions.

4.2.2. Analyze Spatial Data:

One of the strongest features of GIS is its ability to answer spatial questions—such as “What is near this point?”, “How far is location A from location B?”, or “What lies within this boundary?” These questions are critical in fields ranging from emergency response to retail planning.

By running spatial analyses, such as buffer zones or proximity checks, users can evaluate how different geographic elements relate to each other. For instance, GIS can determine the area within a 5 km radius of a fire station or highlight all schools located within a pollution-prone zone.

4.3. Urban Planning, Land Use & Disaster Management

4.3.1. Urban Planning & Land Use:

Geographic Information Systems (GIS) are transformative tools in the realm of urban planning and land use management. They enable city planners to analyze, visualize, and predict how land is used, thereby facilitating more informed and sustainable development decisions.

How City Planners Use GIS

  1. Visualizing Existing Land Use Patterns :
    • Mapping Current Use: Planners can overlay land parcels with current zoning, infrastructure, residential areas, commercial zones, and green spaces.
    • Spatial Analysis: Identifying trends, such as clustering of commercial activities or gaps in public services.
  2. Simulating and Predicting Urban Growth :
    • Growth Models: GIS supports simulations that project how cities may expand based on population trends, economic factors, and policy changes.
    • Scenarios Testing: Planners can visualize the effects of proposed developments or zoning changes before implementation, helping to avoid traffic congestion and overbuilt areas.
  3. Infrastructure Planning :
    • Site Selection: GIS aids in determining optimal sites for new schools, hospitals, parks, or transit corridors by considering:
      • Population density
      • Proximity to existing services
      • Transportation networks
      • Environmental constraints (wetlands, floodplains, conservation areas)
    • Connectivity Assessment: Ensures new facilities are accessible and integrated within the broader city framework.

4.3.2 Disaster Management:

GIS is invaluable during natural disasters—it helps forecast risks, coordinate emergency response, and assess damage afterward. Layers such as elevation, rainfall, and river flow help predict flood-prone zones, while satellite data can track wildfires or landslides in real time.

Rescue teams use GIS to plan evacuation routes, identify shelters, and track resource distribution. After disasters, GIS maps assist in calculating affected areas, helping government and relief agencies respond more effectively and rebuild smarter.

4.4. Agriculture & Crop Monitoring

Modern agriculture uses GIS for what's called precision farming, which means tailoring farming techniques to specific areas of land based on conditions like soil type, moisture levels, and historical yield data.

GIS can help farmers identify which parts of their fields need more water, where pests are concentrated, or where to apply fertilizer. This saves resources, boosts productivity, and makes farming more environmentally friendly.

This approach enables farmers to::

  • Map field conditions: GIS helps create detailed soil and vegetation maps, revealing variability in crop health, soil fertility, and moisture levels across the field
  • Target resource application: Using these maps, farmers can apply water, fertilizers, and pesticides only where needed instead of blanket-treating the whole area, optimizing inputs, saving costs, and reducing environmental impact
  • Boost productivity and sustainability: By precisely targeting interventions, farmers can increase crop yields—sometimes by up to 30%—while minimizing waste and environmental harm
  • Identify problem areas: GIS can highlight spots with poor plant growth, pest hotspots, or nutrient deficiencies, allowing quick and specific remedial action
  • Monitor and plan efficiently: With features like time-series comparison and integration with satellite imagery, farmers can continuously monitor fields and adjust strategies as conditions change

Overall, GIS empowers farmers to use data-driven insights for more efficient, profitable, and environmentally friendly agriculture

4.5. Navigation & Routing

GIS powers nearly all navigation apps, from Google Maps to ride-sharing services. It constantly processes live traffic updates, road closures, and user feedback to recommend the fastest or most fuel-efficient routes.

Logistics and delivery services rely heavily on GIS to optimize routing, reduce travel time, and improve fuel efficiency. Even public transportation planning uses GIS to create better schedules and routes based on commuter demand.

GIS is the backbone of modern navigation and routing, powering everything from everyday apps to complex logistics operations.

  • Navigation Apps: Leading services like Google Maps rely on GIS to provide real-time, turn-by-turn navigation. They process live traffic conditions, road closures, and user-reported incidents to suggest the fastest or most fuel-efficient routes. Users benefit from up-to-date information such as traffic jams or detours and can plan journeys efficiently with tools for walking, driving, cycling, or public transit
  • Logistics & Delivery Optimization: Companies use GIS to optimize routes for delivery vehicles and service fleets, reducing travel time and maximizing fuel efficiency. GIS tools analyze road networks and dynamic conditions (like traffic or weather), enabling intelligent planning for multi-stop routes and efficient response to changing events on the road
  • Public Transportation Planning: City transit agencies utilize GIS to design schedules and routes that match commuter demand, analyzing data on population density, travel patterns, and service usage to improve coverage and efficiency. This results in smarter, more reliable public transport systems.

GIS enables real-time, data-driven decisions for routing and navigation in consumer apps, logistics, and public transportation, making travel faster, more efficient, and responsive to changing conditions.

Welcome to Chapter 5 : How to Start with GIS ? (Even if You’re a Total Beginner)

5.1. Introduction

Feeling overwhelmed? Don't be! Starting with GIS can be fun, free, and simple.

Getting started with GIS might seem overwhelming at first, but it’s easier than you think—especially with today’s beginner-friendly tools and open resources. At its core, GIS is just a way to analyze and visualize geographic data to answer real-world questions. Whether you're interested in mapping your hometown, tracking environmental changes, or planning better cities, GIS gives you the tools to see patterns and make better decisions.

As we have already started by understanding the basic building blocks—Maps, Data, Software, and Spatial Thinking in our previous chapters. Learning what kinds of data go into GIS (like shapefiles or satellite images), what questions GIS can help answer (like "Where should a new school go?"), etc. on our previous chapter, will help in great deal for anyone to confidently enter into this GIS world..

Simple tools like QGIS or Google Earth Engine to get a feel for how GIS works will be explored in this chapter. With free courses, tutorials, and practice datasets available online, you can begin your GIS journey at your own pace—without needing a technical background

5.2. Exploring Free GIS Tools

5.2.1 Google Earth - The Visual Core of GIS:

Go to earth.google.com in your browser or download the Google Earth Pro desktop app for more features.

Google Earth is a free, user-friendly tool that lets you explore the world from your computer or phone. With just a few clicks, you can zoom into your hometown, check out famous landmarks in 3D, or even explore the ocean floor. It’s perfect for beginners who want to get a visual feel of geography, plan trips, or simply enjoy virtual travel. Whether you're curious about a new place or want to view your house from space, Google Earth makes it easy and fun to get started.

5.2.2 QGIS:

What is QGIS? A free, open-source mapping tool (like Google Maps, but way more powerful!) used for land records, planning, environment, and more. It can be freely downloaded from their official website - qgis.org.

I. Why Beginners Love It?

  • Easy to install
  • Clean, simple interface
  • Huge support community + tutorials
  • Works on most computers

II. First Steps Inside QGIS

  • Start by adding layers – shapefiles, satellite images, or OpenStreetMap.
  • Explore your data using the layer panel and map canvas.
  • Use the attribute table to see the data behind the map.

III. What You Can Do Quickly

  • Style maps with different colors or symbols (e.g. population zones).
  • Label features like roads or districts.
  • Measure distances or areas in just a few clicks.

IV. Big Plus

  • You can fully customize your maps and even run basic spatial analysis—without needing to code!

V. Need Help?

  • Tons of videos and tutorials online
  • QGIS docs and forums

Welcome to Chapter 6 : Conclusion: GIS = Real World Superpower

Map = Clark Kent & GIS = SuperMan!

GIS is not just about making maps prettier. It's about thinking spatially, analyzing deeply, and making informed decisions.

  • Maps answers "Where?".
  • GIS goes deeper—it helps answer:
    • "Where?" – Location of things (X,Y coordinates to pinpoint one's precise location on the Earth's surface.)
    • "How?" - How far, how connected, how dense how is it (point or polygon or line, etc)?
    • "What?" - What attributes data for these, What features or data exist there?

But it doesn’t stop there: Once we know where, how, and what, GIS helps us ask the biggest question of all: “Why?” – Why do patterns exist? Why are problems happening there? . It helps to understand about the root cause of the problem and also gives a greater platform to plan for mitigating the problems spatially.

That’s the power of spatial thinking.

In a world filled with complex problems — from urban sprawl to global warming — GIS gives us a smarter, faster, and better way to understand and improve our surroundings.

It’s like giving yourself X-ray vision — but instead of bones, you see the invisible layers of geography, society, and nature interacting!

Hence in the evolving technological advancements and dawn of the age of Artificial Intelligence helps to improve the GIS to manifold and help to generate greater insights to help variety of agencies with their results. This makes the understanding and working on GIS domain a mandatory for many people in order generate greater results in their field.

So next time you open a map app, remember: You are already stepping into the world of GIS!
